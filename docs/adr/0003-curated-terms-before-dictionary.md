# ADR 0003: Curated Terms Before General Dictionary

## Status

Accepted

## Context

Generic dictionaries cannot reliably explain Six Sigma terms in the way a learner needs. For example, "control", "opportunity", and "capability" have generic meanings but specific Six Sigma usage.

## Decision

Lookup order starts with curated Six Sigma phrases and terms, then falls back to general dictionary data.

## Consequences

- Better technical explanations.
- Requires maintaining a term dictionary.
- Unknown words still need a fallback path.
